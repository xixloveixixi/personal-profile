# 阶段游标管理

## 什么是游标

`docs/dev-harness/stage-plan.md` 中，当前正在推进的阶段标题格式为：

```
## 当前阶段：阶段 N [阶段名称]
```

其他已完成阶段的标题格式为：

```
## 阶段 N [阶段名称]（已完成）
```

## 挪动规则

阶段切换时需要同时改两处：
1. 旧阶段：`## 当前阶段：阶段 N ...` → `## 阶段 N ...（已完成）`
2. 新阶段：`## 阶段 N+1 ...` → `## 当前阶段：阶段 N+1 ...`

## AI 的职责

**提醒，不主动执行。**

时机：Gate F 完成后。

提醒话术：
```
本阶段已归档。建议挪动 stage-plan.md 游标：
- 将"当前阶段：阶段 N"改为"阶段 N（已完成）"
- 将下一段"阶段 N+1"改为"当前阶段：阶段 N+1"

需要我帮你改吗？
```

用户确认后，使用 edit_file 执行上述两处修改。

## 验证

挪动后建议跑 `npm run harness:check`，确认报告中阶段名变更正确。
