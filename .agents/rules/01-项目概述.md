# 项目概述

## 项目定位

在个人作品集网站基础上演进为「AI 学习规划与成长记录平台」。

- 公开区面向访客 / 面试官，展示个人信息、项目、Notion 博客、AI 产品入口。
- 私有学习区面向 owner（站长本人），提供学习画像、目标、计划、进度、周复盘等功能，统一采用「左菜单 + 右内容」工作台布局。
- 详细设计见 `tech_design/ai-learning-platform/SDD-AI学习规划与成长记录平台.md`。

## 技术栈

| 领域 | 技术 | 说明 |
|------|------|------|
| 前端框架 | Next.js ^14（App Router） | 强制使用，禁止 Pages Router |
| UI 框架 | React ^18.3 | - |
| 类型系统 | TypeScript ^5.3 | 强制使用，禁止 JavaScript |
| 样式方案 | Tailwind CSS ^3.4 + CSS 变量 | 主样式方案，禁止 SCSS Modules |
| 组件原子 | shadcn / `components/ui` | 原子组件层 |
| 组件库 | Ant Design ^5.29 | 仅用于复杂交互控件二次封装 |
| 动画 | framer-motion ^11 | - |
| 图标 | lucide-react、`@ant-design/icons` | - |
| 状态管理 | Zustand（计划引入） | 客户端全局状态统一使用 |
| AI SDK | `@ai-sdk/react` ^1.2 | Chat 与 Agent 流式交互 |
| 内容 / 博客 | `@notionhq/client`、`notion-to-md` | 博客只读 Notion，不入后端 |
| 数据库（前期） | `@supabase/supabase-js` | 知识库向量与初期辅助存储 |
| 数据校验 | zod ^3.25 | API 入参与 LLM 输出校验 |
| 时间工具 | dayjs（按需） | 涉及日期处理统一 dayjs |
| 后端服务 | Go + Gin + GORM + MySQL + JWT | 由 Harness Stage 1 推进，尚未落地 |

## 关键边界

- 博客内容只来自 Notion，不进入 Go 后端 CRUD。
- 公开数据走 `/api/public/*`，私有学习数据走 `/api/private/*`，所有私有接口需带 JWT。
- 公开区 Chat 不得读取私有学习数据。
- Go 后端代码与表结构推进遵循 `docs/dev-harness/stage-plan.md`。
